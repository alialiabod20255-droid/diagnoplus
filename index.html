Diagnoplus — صفحة التكامل مع Builder.io و Bolt.new

ضع هذه الصفحة في جذر المشروع ليتم اكتشافها واستيرادها في أدوات مثل Builder.io أو Bolt.new

تعليمات سريعة

  1. لـ Builder.io: أنشئ محتوى جديد (Page) في حسابك. استخدم إما SDK أو iframe لعرض المحتوى داخل هذه الصفحة.
  2. لـ Bolt.new: يمكنك استيراد هذا المستودع من GitHub في واجهة bolt.new، وسيبحث عن ملف index.html في الجذر ليعرضه أو يستخدمه كنقطة بداية.
  3. بعد رفع الملف للمستودع، افتح Builder.io → models أو content وحدد preview URL أو استخدم iframe المعروض أدناه.

مثال: عرض صفحة Builder.io عبر iframe (بسيط وعملي)

استبدل YOUR_BUILDER_PAGE_URL بعنوان المعاينة/الصفحة من Builder.io أو استخدم URL الذي يعطيه Builder لمحتواك.

ملاحظة: إذا كنت تستخدم طريقة الـ SDK أو webcomponents فستحتاج لإضافة سكربت CDN الرسمي الخاص بـ Builder قبل استخدام <builder-component>.

مثال: جلب محتوى Builder عبر REST API (مثال عملي)

استبدل YOUR_BUILDER_API_KEY وYOUR_PATH بالقيم الحقيقية. هذا مثال بسيط يري كيفية جلب JSON وعرض الـ HTML إن وُجد.

<script>
  async function renderBuilderContent() {
    const apiKey = 'YOUR_BUILDER_API_KEY'; // ضع مفتاح القراءة هنا
    const url = 'https://cdn.builder.io/api/v1/content/page?apiKey=' + apiKey + '&url=/';
    try {
      const res = await fetch(url);
      const json = await res.json();
      // هذا مجرد مثال: محتوى Builder قد يطلب تحويله إلى HTML باستخدام builder sdk أو webcomponents
      document.getElementById('builder-json-output').textContent = JSON.stringify(json, null, 2);
    } catch(e) {
      console.warn('Builder fetch failed', e);
    }
  }
  renderBuilderContent();
</script>
محتوى Builder سيظهر هنا بعد استدعاء الـ API (استبدل المفاتيح أولاً).

ماذا عن Bolt.new؟

Bolt.new (bolt.new) يمكنه استيراد مستودع GitHub وإنشاء تطبيق/صفحة انطلاق من ملف الجذر مثل index.html. بعد رفع هذا الملف إلى المستودع:

  • افتح https://bolt.new → اختر "Import from GitHub" → أدخل رابط المستودع → سيحاول العرض التلقائي تشغيل الملف الجذري.
  • تأكد أن هذا الملف مستقل قدر الإمكان (CSS صغير/inline أو روابط CDN) ليعرض جيد داخل بيئة Bolt.
للمساعدة: استبدل قيم YOUR_BUILDER_API_KEY و YOUR_PAGE_ID و YOUR_BUILDER_PAGE_URL بالقيم الفعلية. وإذا رغبت أعد كتابة الصفحة لتعمل مع WebComponents الرسمي لـ Builder.io - أرسل لي وسأجهز لك النسخة.
